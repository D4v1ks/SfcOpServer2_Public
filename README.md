## The 2nd interaction of the 'Starfleet Command 2 - Orion Pirates' multiplayer server

### What it contains:

* The server console project, used to launch the multiplayer campaign service, and the other services needed by the game
* The client console project used to launch the game, setup things, and connect the user to the server
* The 'Met\_Common' script required for everything to work

### Assets required:

* To play a campaign by yourself, or with your friends in a lan party, you will need a compatible game:
[More on this later...](https://hotandspicyforums.com)
* To edit the campaign map used by the server you will need a compatible tool:
[CampaignMapEditorFlipped.7z](https://1drv.ms/u/c/5bea448f9325a62b/Edzsfg3cpz9Hpa8qnS-TwpMBITgsEHpcwKBCdDM1RR01DQ)
